# Debuggers y herramientas de reversing

Para analizar el comportamiento interno de las aplicaciones se utilizaron herramientas de debugging y reversing.

## Ghidra

Herramienta de ingeniería inversa utilizada para análisis estático de binarios.

## IDA

Disassembler utilizado para analizar el código de los programas.

## Immunity Debugger

Debugger utilizado para analizar aplicaciones vulnerables durante su ejecución.
