# Lenguajes de programación

Durante la preparación del entorno de laboratorio se instalaron diferentes lenguajes de programación que permiten ejecutar herramientas de análisis y desarrollar scripts utilizados durante el proceso de explotación.

En el ámbito de la seguridad ofensiva, los lenguajes de programación cumplen un papel fundamental, ya que permiten automatizar tareas, interactuar con aplicaciones vulnerables y desarrollar pruebas de concepto que demuestran la explotación de una vulnerabilidad.

## Python

Python es uno de los lenguajes más utilizados dentro del campo de la seguridad informática y el desarrollo de exploits.

Se utiliza ampliamente para desarrollar scripts que interactúan con servicios vulnerables, enviar payloads a aplicaciones que reciben datos por red y automatizar diferentes fases del proceso de explotación.

Entre sus principales usos dentro del laboratorio destacan:

- Desarrollo de scripts de explotación
- Automatización de tareas repetitivas
- Envío de datos maliciosos a servicios vulnerables
- Desarrollo de pruebas de concepto (Proof of Concept)

La simplicidad de su sintaxis y la gran cantidad de librerías disponibles hacen que Python sea una herramienta extremadamente flexible para investigadores de seguridad.

## Java

Java es necesario para ejecutar algunas herramientas de ingeniería inversa utilizadas durante el laboratorio.

Una de las herramientas más importantes que utiliza este lenguaje es **Ghidra**, una herramienta de reversing desarrollada por la NSA que permite analizar binarios, desensamblar código y comprender el funcionamiento interno de una aplicación.

Por este motivo, la instalación de Java resulta imprescindible para poder ejecutar herramientas de análisis estático utilizadas durante el proceso de investigación de vulnerabilidades.
