# Entornos de desarrollo

Durante la preparación del laboratorio se instalaron diferentes entornos de desarrollo y editores de código que permiten trabajar de forma cómoda con scripts, revisar código fuente y desarrollar herramientas utilizadas durante el proceso de análisis de vulnerabilidades.

Disponer de un entorno de desarrollo adecuado resulta fundamental, ya que facilita la escritura de código, la organización del proyecto y la depuración de scripts utilizados durante la investigación.

Las herramientas utilizadas en este entorno fueron las siguientes.

## Notepad++

Notepad++ es un editor de texto ligero ampliamente utilizado para editar archivos de código y configuraciones.

Entre sus ventajas destacan:

- Ligereza y rapidez de ejecución
- Soporte para múltiples lenguajes de programación
- Resaltado de sintaxis
- Edición rápida de scripts

En el laboratorio se utilizó principalmente para modificar scripts simples y revisar archivos de configuración.

## Visual Studio Code

Visual Studio Code es uno de los entornos de desarrollo más utilizados actualmente. Se trata de un editor muy flexible que permite trabajar con múltiples lenguajes de programación mediante extensiones.

Entre sus características destacan:

- Soporte para múltiples lenguajes
- Integración con Git
- Amplia variedad de extensiones
- Depuración integrada

Visual Studio Code se utilizó para desarrollar scripts en Python utilizados durante el proceso de explotación.

## PyCharm

PyCharm es un entorno de desarrollo especializado en Python. Está diseñado para facilitar el desarrollo de aplicaciones complejas en este lenguaje.

Entre sus funcionalidades destacan:

- Autocompletado inteligente
- Herramientas de depuración
- Gestión de proyectos Python
- Integración con entornos virtuales

En el laboratorio se utilizó principalmente para desarrollar scripts utilizados en la interacción con aplicaciones vulnerables. PyCharm
