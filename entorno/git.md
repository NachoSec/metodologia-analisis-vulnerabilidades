# Control de versiones

## Git

Git es un sistema de control de versiones ampliamente utilizado para gestionar proyectos de software y documentación técnica.

En el contexto de este laboratorio se utilizó Git junto con la plataforma GitHub para organizar la documentación del proyecto y publicar el trabajo realizado.

El uso de Git permite:

- Mantener un historial de cambios en el proyecto
- Gestionar versiones del trabajo realizado
- Organizar la documentación de forma estructurada
- Compartir el proyecto con otros usuarios

## GitHub

GitHub es una plataforma que permite alojar repositorios Git y compartir proyectos de forma pública o privada.

En este proyecto se utilizó GitHub para publicar la documentación del laboratorio, organizar la estructura del repositorio y facilitar el acceso al trabajo realizado.

La utilización de Git y GitHub permite presentar el trabajo de forma profesional y estructurada, algo habitual en proyectos técnicos dentro del ámbito de la seguridad informática.
